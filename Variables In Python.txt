What is a Variable in Python?
A variable in Python is a name given to a memory location that is used to store data (values) so that it can be used and changed later in a program.
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
In simple words:
A variable is like a container that holds some data.
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Why Variables Are Needed?
Variables help us to:
  1. Store information
  2. Reuse values
  3. Change data during program execution
  4. Make programs readable and meaningful
Without variables, programs would become confusing and difficult to manage.
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Real-Life Example
Think of a labeled box:
  1. The label = Variable name
  2. The item inside the box = Value
Example idea:
  1. A box labeled contains the value 25
  2. A box labeled name contains the value "Rahul"
You can change the item inside the box, but the label remains the same.
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
How Variables Work in Python
When you create a variable:
  1. Python creates memory space
  2. Stores the value in memory
  3. The variable name points to that memory location
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Text-Based Diagram
Variable Name  →  Memory Location  →  Value
     age       →     0x101A        →   25
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Creating a Variable in Python
In Python, you create a variable by assigning a value using =

Example:
x = 10

Explanation:
x → variable name
= → assignment operator
10 → value stored
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Important Rules for Naming Variables
  1. Variable name must start with a letter (a–z, A–Z) or underscore (_)
  2. It cannot start with a number
  3. Only letters, numbers, and underscore are allowed
  4. Variable names are case-sensitive
        age and Age are different
  5. Do not use keywords (reserved words)

Good vs Bad Variable Names
Good Names
  1. age
  2. total_marks
  3. student_name
Bad Names
  1. 1age ❌
  2. total-marks ❌
  3. class ❌ (keyword)
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Changing the Value of a Variable
Variables can be updated:
x = 10
x = 20
Now x stores 20, not 10.
