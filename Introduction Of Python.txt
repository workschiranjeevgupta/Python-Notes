What is Python?
Python is a high-level, interpreted, general-purpose programming language that is used to create software, websites, applications, games, and to work with data, artificial intelligence, and automation.
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
In simple words:
Python is a language that helps humans communicate instructions to a computer in an easy and readable way.
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Why Python Was Created?
Python was created to solve a major problem in programming:
  1. Older languages were hard to read
  2. Syntax was complex
  3. Writing programs took more time
Python was designed to be:
  1. Easy to learn
  2. Easy to read
  3. Easy to write
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Key Characteristics of Python
1. Simple and Readable
  1.1. Python code looks almost like English sentences.
  1.2. Example idea (no code yet):
       “Print this message” is written almost the same way in Python.
2. High-Level Language
  2.1. You do not need to manage memory or hardware.
  2.2. Python handles complex tasks internally.
3. Interpreted Language
  3.1. Python runs code line by line.
  3.2.Errors are shown immediately, making debugging easier.
4. General-Purpose
  4.1. One language can be used for many purposes:
    4.1.1. Web development
    4.1.2. Software development
    4.1.3. Data analysis
    4.1.4. Artificial Intelligence
    4.1.5. Automation
5. Free and Open Source
  5.1. Anyone can download and use Python without paying.
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
Where Python Is Used?
Python is widely used in many fields:
1. Web Development – websites and web apps
2. Data Science – analyzing large amounts of data
3. Artificial Intelligence & Machine Learning
4. Automation – automating repetitive tasks
5. Game Development
6. Desktop Applications
---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
How Python Works (Flow Explanation)
Think of Python working in this flow:
1. You write Python code
2. Python Interpreter reads the code
3. Interpreter executes code line by line
4. Computer performs the task
5. Output is displayed
